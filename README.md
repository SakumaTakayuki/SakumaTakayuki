## 📘 ポートフォリオ

### 🔹 **CSV → Web登録 自動化ツール**
CSV を読み込み、Webフォームへ自動入力し、結果をログとして保存するツールです。

- Streamlit ベースの UI  
- Selenium によるブラウザ自動操作  
- SQLAlchemy + SQLite によるログ管理  
- pytest で単体テスト完備  
- メモリDBを使った高速テストも実装
